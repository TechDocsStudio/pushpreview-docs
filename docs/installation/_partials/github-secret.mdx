To configure the GitHub secret, follow these steps:

1. Generate a PushPreview token. For more information, see [Authentication](/configuration/authentication).

1. Go to the **Settings** tab of your GitHub repository or organization.

1. In the **Settings** menu, navigate to **Secrets** and chose **Actions**.

1. Click on **New repository** secret or **New organization secret**, depending on your choice in the previous step.

1. Use the following details to create a new token:

    * **Name**: `PUSHPREVIEW_TOKEN`
    * **Token**: The token you have generated.

1. Click **Add secret**.
